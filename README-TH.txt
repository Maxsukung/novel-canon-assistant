NOVEL STUDIO V1 — วิธีติดตั้งบน iPad

1) สร้าง GitHub Repository แบบ Public และไม่ต้องเพิ่ม README
2) แตก ZIP แล้วอัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ให้ index.html อยู่หน้าแรกของ Repository
3) ไป Settings > Pages
4) Source: Deploy from a branch / Branch: main / Folder: /(root)
5) เปิด URL ที่ GitHub Pages แสดงด้วย Safari
6) กด Share > Add to Home Screen
7) เปิดครั้งแรกขณะออนไลน์ หลังจากนั้นใช้งานส่วนหลักแบบออฟไลน์ได้

ข้อมูลทั้งหมดเก็บใน IndexedDB ของ Safari ควรส่งออกไฟล์สำรองเป็นระยะ
PDF ที่เป็นภาพสแกนล้วนยังอ่านไม่ได้ เพราะไม่มี OCR
